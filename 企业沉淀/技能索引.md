## 使用方式

### 如果你自己用
直接在 Obsidian 里打开可读版阅读

### 如果你想分享给朋友
打开对应 skill 的原始文件/目录，分享即可。对方如果也用 Hermes，把文件放进 `~/.hermes/skills/` 或 `/opt/data/skills/`，Hermes 自动识别加载。

### 更新规则
- 每次创建新 skill → 立即写可读版 + 原始 skill 到 vault
- 更新 skill 时 → 顺手同步 vault 版本
- 索引页随新 skill 自动追加
